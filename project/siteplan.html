Project Site Plan

Site Name

"Who Goes First" The name highlights the main purpose of the website in a straightforward and catchy way.

Site Purpose

The site offers multiple options or tools that is used to determine a player's turn order in any game that requires turns.

  • Coin Toss: For two players. Each player tosses a coin, saves the results, and displays the turn order based on the results.
  • Dice Roll: For any amount of players. Each player rolls a dice, saves the results, and displays the turn order based on the number rolled on the dice.
  • Random Turn: For any amount of players. Each player will randomly receive a number that will determine their turn order.

Scenarios

  • How can I determine the turn order randomly without bias?
  • How can I automatically determine the turn order without doing the calculations myself?
